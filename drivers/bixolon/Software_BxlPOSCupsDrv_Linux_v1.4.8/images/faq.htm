1.How many files are in BIXOLON CUPS Driver Installation CD and what are they?

The BIXOLON CUPS Driver Installation CD contains 5 files and 3 folder that are
            rastertoBixolon_v1.x.x, SoftwareLicense.txt, setup_v1.x.x.sh, readme.htm, uninstaller_v1.x.x.sh, usb and two folders called images and Bixolon, CUPS1.7.2



2. Could you please expalin about the files in the BIXOLON CUPS Driver Installation CD?

rastertoBixolon_v1.x.x
            CUPS Driver for BIXOLON Printer
setup_v1.x.x.sh
            Is a shell script file used to install the BIXOLON CUPS Driver in console mode
readme.htm
            Is a html based user manual for BIXOLON CUPS Driver
uninstaller_v1.x.x.sh
            Is a shell script file used to uninstall the BIXOLON CUPS Driver in console mode
images
            This folder contains some images and html files, which are useful for readme.htm file
Bixolon
            Postscript Printer Description file for BIXOLON Printer



3. I'm Very new to CUPS and is the first time i'm going to install BIXOLON CUPS Driver. What will i do first?

If you are very new to CUPS. Kindly go to http://127.0.0.1:631/help/ This will give you overall information about CUPS
if you are installing BIXOLON CUPS Driver please refer readme.htm carefully before begins installation.

5.How do I check whether BIXOLON CUPS Driver successfully installed in my Linux Distribution?

          Run the following Commands
                    ls /usr/share/cups/model/Bixolon
                    ls /usr/bin/rastertoBixolon_v1.x.x
                    ls /usr/lib/cups/filter/rastertoBixolon_v1.x.x
Successful returns of all 3 commands will indicate successful installation of BIXOLON CUPS Driver



6.BIXOLON CUPS Driver printer setting through Open Office is not taking effect. What could be the reason?

This problem is not related with our driver, is related with the linkage between the Open Office and CUPS. Since our driver is based on CUPS, whatever the setting you made through Open Office will override by CUPS admin. So please use CUPS Admin tool to change the printer settings.
