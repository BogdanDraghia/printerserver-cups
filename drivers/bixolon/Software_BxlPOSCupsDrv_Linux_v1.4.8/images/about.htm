ABOUT BIXOLON CUPS DRIVER

About This Package:

                  The 'BIXOLON CUPS DRIVER printer driver was developed to add printing support for BIXOLON Thermal printer under Linux operating systems, using the widely available and popular CUPS (Common UNIX Printing System) printing system. The driver was written using the C programming language, and utilizes various CUPS API (application programming interface) calls. This driver will enable users of BIXOLON printers to print their receipt under the Linux operating systems.

Supported Operating System:

Operations in an unsupported environment are not guaranteed.

-openSUSE 11.3 32/64bit

-Red Hat Enterprise Linux 7.3 64bit

-CentOS 6.6 32/64bit

-Ubuntu 10.04 LTS 32/64bit

Assumption:

                  This BIXOLON CUPS DRIVER Package assuming that the following packages are already installed in your system:

                        1. CUPS server & architecture (see www.cups.org)
                        2. cups and cups-devel packages
                        3. ghostscript library packages
